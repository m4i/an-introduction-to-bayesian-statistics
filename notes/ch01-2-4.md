## 例1.4: サイコロの丁半（有限の場合の小さな例）

$\Omega = \lbrace 1, 2, 3, 4, 5, 6\rbrace$ で「偶数か奇数か」だけ気にする問題:

- σ加法族: $\mathcal{F} = \lbrace \emptyset, E, E^c, \Omega\rbrace$（ $E = \lbrace 2, 4, 6\rbrace$ = 偶数）
- 確率: $P(\emptyset) = 0,  P(E) = P(E^c) = 1/2,  P(\Omega) = 1$

→ 有限の場合でも、 $2^\Omega$ より小さなσ加法族で十分なことを示す例。

## 例1.5: 線分上にランダムに一点を選ぶ（測度論の本領）

$\Omega = [0, 1]$ で、区間 $[a, b] \subset [0, 1]$ を選ぶ確率を「長さ $b - a$」にしたい。

#### σ加法族の取り方

- $[0, 1]$ に含まれる **すべての閉区間を元に持つ最小のσ加法族** を $\mathcal{F}$ とする
- これを **ボレル集合族** と呼び、後で $\mathcal{B}(\mathbb{R})$ と書く（次節）

#### σ加法族の "元" って何？（具体例）

サイコロ $\Omega = \lbrace 1,...,6\rbrace$ では「 $E = \lbrace 2, 4, 6\rbrace$ が元」のように具体的にわかったが、 $\Omega = [0,1]$ では元が何かピンと来にくい。
**σ加法族の元 = $\Omega$ の部分集合** なので、 $\mathcal{F}$ の元は $[0,1]$ の部分集合。代表的なもの:

| 元の例 | 表記 |
|---|---|
| 空集合 | $\emptyset$ |
| 一点 | $\lbrace 0.5\rbrace$ |
| 閉区間 | $[0.3, 0.7]$ |
| 開区間 | $(0.3, 0.7)$ |
| 有限個の点 | $\lbrace 0.1, 0.5, 0.8\rbrace$ |
| 有理数全体 | $\mathbb{Q} \cap [0, 1]$ |
| 無理数全体 | $[0, 1] \setminus \mathbb{Q}$ |
| 全体 | $[0, 1]$ |

これら全部が「ボレル集合族の元」。閉区間から出発して、補集合・可算和・可算交わりを繰り返すと、これだけ多様な部分集合が自動的に含まれる。

#### 閉区間の補集合は閉区間じゃない（半開区間や和になる）

$\Omega = [0, 1]$ における $[a, b]$ の補集合は、位置によって以下のようになる:

| 閉区間 $[a, b]$ | 補集合 |
|---|---|
| $[0, 0.5]$（左端ぴったり） | $(0.5, 1]$（右側半開区間） |
| $[0.5, 1]$（右端ぴったり） | $[0, 0.5)$（左側半開区間） |
| $[0.3, 0.7]$（内側） | $[0, 0.3) \cup (0.7, 1]$（半開区間2つの和） |
| $\lbrace 0.5\rbrace  = [0.5, 0.5]$（一点） | $[0, 0.5) \cup (0.5, 1]$（1点を抜いた残り） |
| $[0, 1]$（全体） | $\emptyset$ |

つまり「閉区間の補集合を全部含む」 = **半開区間** や **半開区間の和** を全部含む。

#### こうして色々な部分集合が作れる

σ加法族は補集合と可算和（→ 可算交わりも）で閉じているので、閉区間から次々に作れる:

| 作りたい集合 | 構成 |
|---|---|
| 半開区間 $(a, b]$ | $[a, b] \setminus \lbrace a\rbrace  = [a, b] \cap \lbrace a\rbrace ^c$ |
| 開区間 $(a, b)$ | $[a, b] \setminus (\lbrace a\rbrace  \cup \lbrace b\rbrace )$ |
| 一点 $\lbrace a\rbrace$ | 閉区間 $[a, a]$ |
| 有限個の点 | 一点集合の有限和 |
| 有理数全体 $\mathbb{Q} \cap [0,1]$ | 一点集合の **可算和**（有理数は可算個） |
| 無理数全体 | 有理数の補集合 |

「閉区間 + 補集合 + 可算和」の3つだけで、自然に欲しい部分集合がほぼ全部作れる、というのがσ加法族の威力。

> **「最小」と言うからには複数ある**: 「閉区間を全部含むσ加法族」は実は複数あって、その中で一番小さいものを選んでいる。詳しくは下の補足を参照。

#### 確率測度

任意の閉区間 $[a, b] \in \mathcal{F}$ に対し $P([a, b]) = b - a$ と定めれば、これは定義 1.3 を満たす確率測度になる。
これがいわゆる **ルベーグ測度** の制限版。

#### ここがポイント

- $\mathcal{F}$ は $2^{[0,1]}$ より **真に狭い**（病的な部分集合は除外されている）
- でも、私たちが普通に考えたい部分集合は全部 $\mathcal{F}$ に入っている
- だから **有理数 vs 無理数のパラドックス** に巻き込まれずに済む

→ σ加法族の制限 = 「素朴に確率を考えると壊れる集合」を最初から排除する仕組み。

## 補足: なぜ「閉区間を含む最小のσ加法族」と書くのか

「最小の」と言う以上、**複数あって一番小さいものを選んでいる** ということ。なぜ複数あるのか、なぜ最小に意味があるのかを整理する。

### なぜ「閉区間を含むσ加法族」が複数あるのか

「閉区間をすべて含む」は **制約** であり、その制約を満たすσ加法族は無数にある。

- **最大**: $2^{[0,1]}$（冪集合）。すべての部分集合を含むので閉区間も当然含む
- **最小**: ボレル集合族 $\mathcal{B}([0,1])$。閉区間から補集合・可算和を最小限繰り返してできる
- **中間**: ルベーグ可測集合族 $\mathcal{L}([0,1])$。ボレル集合族に「測度0の集合の任意の部分集合」を加えた拡張版

包含関係:

$$
\mathcal{B}([0,1]) \ \subsetneq\  \mathcal{L}([0,1]) \ \subsetneq\  2^{[0,1]}
$$

すべて「閉区間を全部含むσ加法族」だが、サイズが違う。

### なぜ「最小」が一意に定まるのか

σ加法族の集合族について、**共通部分（共通する元のみで作る族）はそれ自身σ加法族** になる（標準的な事実）。

なので「閉区間を含むσ加法族すべて」の共通部分を取ると:

- それ自身がσ加法族
- 閉区間を全部含む
- 元のどんな「閉区間を含むσ加法族」よりも小さい

→ これが「最小」として一意に定まる。これを **「閉区間によって生成されるσ加法族」** と呼び、ボレル集合族の定義になる。

### なぜ最小を採るのか

直感的には「閉区間さえ含んでいれば十分。余計なものは入れたくない」。

- 大きすぎるσ加法族（極端には $2^{[0,1]}$）を取ると、確率測度を整合的に定義できない部分集合まで含んでしまう（1.2.2 の有理数 vs 無理数のパラドックス）
- 必要十分なところで止めたい → **最小** を選ぶ

「素朴に欲しい部分集合（区間とその組み合わせ）はカバー」しつつ「病的な部分集合は排除」する、というバランスが取れた選び方がボレル集合族。

### 冪集合に「追加で入っている」のは何か

ボレル集合族と冪集合の差は、**選択公理を使ってのみ構成できる病的な集合**。代表例が **Vitali 集合**。

#### ボレル集合族にはほとんどの「自然な」集合が入っている

閉区間を起点に補集合・可算和（→ 可算交わりも）を許すと、言葉で書き下せる集合はほぼ全部含まれる:

| 部分集合 | ボレル集合族での作り方 |
|---|---|
| 開区間 $(a, b)$ | $[a, b] \setminus (\lbrace a\rbrace  \cup \lbrace b\rbrace )$ |
| 一点 $\lbrace a\rbrace$ | 閉区間 $[a, a]$ |
| 有限個の点 | 一点集合の和 |
| 有理数全体 $\mathbb{Q} \cap [0, 1]$ | 可算個の一点集合の和（有理数は可算） |
| 無理数全体 | 有理数の補集合 |
| カントール集合 | 閉区間の可算交わり |

#### Vitali 集合の作り方（選択公理が要る）

1. $[0, 1]$ 上で「2つの数の差が有理数」という関係を定義（同値関係）
2. これにより $[0, 1]$ が無数の同値類に分割される（各類は可算無限）
3. **選択公理** を使って各同値類から代表元を1つずつ取り出す
4. 代表元の集合 $V$ が **Vitali 集合**

#### Vitali 集合の何が病的か

$V$ に対しては **どんな測度を入れても整合的に確率を割り当てられない**:

- $V$ を有理数 $q \in [0, 1] \cap \mathbb{Q}$ だけずらした集合 $V_q$ を考えると、 $\lbrace V_q\rbrace _q$ は **可算個** で互いに素、その和で $[0, 1]$ を覆える
- 平行移動で形は変わらないので $P(V) = P(V_q)$ のはず
- 可算加法性より $P([0, 1]) = \sum_q P(V_q) = \sum_q P(V)$
- $P(V) = 0$ なら $P([0,1]) = 0$ で矛盾、 $P(V) > 0$ なら無限大で矛盾

→ 確率も長さも与えられない、まさに「病的」な集合。

#### 結論

| 集合族 | 含まれるもの |
|---|---|
| ボレル集合族 $\mathcal{B}$ | 自然に書き下せるほぼすべての集合 |
| ルベーグ可測集合族 $\mathcal{L}$ | ボレル + 測度0の集合の任意の部分集合（Vitali は含まない） |
| 冪集合 $2^{[0,1]}$ | あらゆる部分集合（Vitali 集合などを含む） |

冪集合を確率の定義域にすると Vitali 集合のような **相手にしたくないものまで含めてしまい**、整合的な確率が定義できなくなる。
ボレル集合族は「自然に欲しいものは全部、病的なものは排除」というバランスで「ちょうどいい」サイズを実現している。これが「最小」を選ぶ実質的な意味。
