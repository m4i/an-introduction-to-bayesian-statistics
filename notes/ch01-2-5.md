# 1.2.5 確率変数とは

> 該当ページ: 20–22（[book/0030.jpg](../book/0030.jpg)–[book/0032.jpg](../book/0032.jpg)）

## 3行サマリ

1. 確率変数 $X : \Omega \to S$ は確率空間から値の空間への **可測な写像**
2. 条件: 任意の $G$ に対し $X^{-1}(G) \in \mathcal{F}$（ $X$ の値の集合の逆像が常にσ加法族の元）
3. 実数値の場合は $\lbrace X \le a\rbrace  \in \mathcal{F}$ だけ要請すれば十分（定義1.7）

ベイズ統計の主役 **確率変数** をきちんと定義する。

## 直感

確率変数とは、確率空間 $\Omega$ の各点 $\omega$（"確率世界の状態"）に、観測される値（数値・ベクトルなど）を対応させる **写像**。

### 値は「数」である必要はない（が、実用上はほぼ数）

定義1.6（一般）では行き先 $S$ は **任意の可測空間** でよく、必ずしも数値でない:

| 行き先 $S$ | 例 |
|---|---|
| カテゴリ | $X : \Omega \to \lbrace \text{赤}, \text{青}, \text{緑}\rbrace$ |
| 文字（数に変換しない） | $X : \Omega \to \lbrace \text{表}, \text{裏}\rbrace$ |
| 実数 | $X : \Omega \to \mathbb{R}$（最も一般的） |
| ベクトル | $X : \Omega \to \mathbb{R}^n$ |

定義1.7 はそのうち **実数値だけに限定** したもの。本書も「実数値で十分な場合が多い」と書いている。理由:
- 数の方が扱いやすい（足し算・掛け算、平均・分散）
- 観測量は大抵数値（身長、温度、個数、時間 ...）
- 非数値もほぼ数に符号化される（カテゴリ → one-hot、表裏 → 0/1）

なので「元を **数（または別の何か）** に変換したものが確率変数」という直感は基本的に正しい。

### なぜわざわざ確率変数に変換するのか

サイコロのような単純な例だけなら、 $\Omega = \lbrace 1,...,6\rbrace$ をそのまま使えば足りる。確率変数を使う本当の利点は次の4つ:

#### ① $\Omega$ の元は数とは限らない（計算するには数が必要）

期待値や分散は値が数であることを要求する:

$$
E[X] = \sum_x x \cdot P(X = x)
$$

コイン投げで $\Omega = \lbrace \text{表}, \text{裏}\rbrace$ をそのまま使うと「**表 × 確率**」が意味をなさない。確率変数で 0/1 などに変換して初めて期待値が計算できる。

#### ② 同じ $\Omega$ から複数の観測量を取り出せる

例: サイコロ2回投げ $\Omega = \lbrace (i,j)\rbrace$ から、

| 観測量 | 確率変数 |
|---|---|
| 1回目の目 | $X(i, j) = i$ |
| 2回目の目 | $Y(i, j) = j$ |
| 合計 | $S(i, j) = i + j$ |
| 最大値 | $M(i, j) = \max(i, j)$ |

がすべて同じ $\Omega$ 上で並列に取り出せる。

#### ③ 確率変数の関数が新しい確率変数になる

$X, Y$ が確率変数なら $X + Y$, $X^2$, $|X - Y|$ などもすべて確率変数。**値の空間（数）で計算ができる** からこそ可能。 $\Omega$ の元（抽象オブジェクト）同士を足したり掛けたりはできない。

#### ④ $\Omega$ の元は「直接観測できない」ことが多い

実問題では $\omega$ は「世界の真の状態」を表す抽象オブジェクトで、観測できるのは $X(\omega)$ だけ:

- ベイズ推定: $\omega$ = コインの真偽 + 投げた結果。観測可能なのは投げた結果のみ
- 統計: $\omega$ = 真の母集団 + サンプル結果。観測可能なのはサンプル値

→ **「世界 $\omega$ → 観測 $X(\omega)$」という写像** が確率変数の本質。

#### 「元をそのまま使う」は恒等写像という特殊ケース

サイコロ $X(\omega) = \omega$ は恒等写像の確率変数で、これは「元をそのまま使う」と同じ。なので **ダメではない、ただ単純すぎる場合に対応する特殊ケース** に過ぎない。

一般には上の4つの理由から、 $\Omega$ と確率変数を分けて考えるのが標準。

### 比喩

- **$\Omega$**: 世界の地図（どんな世界がありうるかの全体）
- **$\omega \in \Omega$**: 今いる地点（世界の状態）
- **確率変数 $X$**: 地点ごとに「気温」「標高」「人口」を返す関数
- **$X(\omega)$**: 今いる地点での気温

地点 $\omega$ そのものより「気温 = 25℃」の方が扱いやすいし、同じ地図上に複数の観測量を並列に載せられる、という構図。

## 具体例

### 例1: コイン投げ（最小例）

**確率空間**:
- $\Omega = \lbrace \text{表}, \text{裏}\rbrace$
- $\mathcal{F} = 2^\Omega$
- $P(\lbrace \text{表}\rbrace ) = P(\lbrace \text{裏}\rbrace ) = 1/2$

**確率変数** $X$（表なら1、裏なら0）:
$$X(\text{表}) = 1, \qquad X(\text{裏}) = 0$$

「 $X$ が 1 を取る確率」は $P(X = 1) = P(\lbrace \omega : X(\omega) = 1\rbrace ) = P(\lbrace \text{表}\rbrace ) = 1/2$。

### 例2: サイコロ — 同じ $\Omega$ に複数の確率変数

**確率空間**:
- $\Omega = \lbrace 1, 2, 3, 4, 5, 6\rbrace$
- $\mathcal{F} = 2^\Omega$
- $P(\lbrace i\rbrace ) = 1/6$

**3つの確率変数を同じ $\Omega$ 上に定義**:

| 確率変数 | 定義 | 意味 |
|---|---|---|
| $X(\omega) = \omega$ | 恒等写像 | 出目そのもの |
| $Y(\omega) = \omega \bmod 2$ | 偶奇 | 偶数なら 0、奇数なら 1 |
| $Z(\omega)$（下式） | 6 で 100 | 「6 が出たら 100 円もらえる」ゲーム |

$Z$ は次の通り:

$$
Z(\omega) = \begin{cases} 100 & (\omega = 6) \\\\ 0 & (\text{その他}) \end{cases}
$$

**確率の計算**:
- $P(X \ge 4) = P(\lbrace 4, 5, 6\rbrace ) = 1/2$
- $P(Y = 1) = P(\lbrace 1, 3, 5\rbrace ) = 1/2$
- $P(Z = 100) = P(\lbrace 6\rbrace ) = 1/6$

→ **同じ $\Omega$ から、注目する側面に応じていろんな確率変数が作れる**（1.2.6 の哲学）。

### 例3: 2回コイン投げ — 独立性の例

**確率空間**:
- $\Omega = \lbrace HH, HT, TH, TT\rbrace$
- $\mathcal{F} = 2^\Omega$
- $P(\lbrace HH\rbrace ) = P(\lbrace HT\rbrace ) = P(\lbrace TH\rbrace ) = P(\lbrace TT\rbrace ) = 1/4$

**確率変数**:

| 確率変数 | 定義 |
|---|---|
| $X$ = 表が出た回数 | $X(HH)=2,\ X(HT)=X(TH)=1,\ X(TT)=0$ |
| $Y$ = 1回目が表なら 1 | $Y(HH)=Y(HT)=1,\ Y(TH)=Y(TT)=0$ |
| $Z$ = 2回目が表なら 1 | $Z(HH)=Z(TH)=1,\ Z(HT)=Z(TT)=0$ |

**$X$ の確率分布**: $P(X=0)=1/4,\ P(X=1)=1/2,\ P(X=2)=1/4$。

**$Y$ と $Z$ は独立**: $P(Y=1, Z=1) = P(\lbrace HH\rbrace ) = 1/4 = (1/2)(1/2) = P(Y=1)P(Z=1)$。

### 例4: $[0,1]$ 上の一点（連続値）

**確率空間**:
- $\Omega = [0, 1]$
- $\mathcal{F} = $ ボレル集合族
- $P([a, b]) = b - a$

**確率変数**:

| 確率変数 | 定義 | 意味 |
|---|---|---|
| $X(\omega) = \omega$ | 恒等写像 | $[0,1]$ 上の一様分布 |
| $Y(\omega)$（下式） | 左半分なら 1 | インジケータ |

$Y$ は次の通り:

$$
Y(\omega) = \begin{cases} 1 & (\omega \le 0.5) \\\\ 0 & (\omega > 0.5) \end{cases}
$$

**確率の計算**:
- $P(X \le a) = P([0, a]) = a$
- $P(Y = 1) = P([0, 0.5]) = 0.5$

### 共通する構造

どの例でも、 $P(X \in A)$ は **$\Omega$ 側の事象 $X^{-1}(A) = \lbrace \omega : X(\omega) \in A\rbrace$ の確率** として計算される:

$$
P(X \in A) = P(X^{-1}(A))
$$

「 $X$ が値の集合 $A$ に入る確率」を知りたければ、 $\Omega$ に戻って **その値が出るような $\omega$ の集合** の確率を見ればよい、ということ。

## 定義

> **定義 1.6** 確率空間 $(\Omega, \mathcal{F}, P)$ から可測空間 $(S, \mathcal{G})$ への写像 $X : \Omega \to S$ で、任意の $G \in \mathcal{G}$ について
> $$\lbrace \omega \in \Omega : X(\omega) \in G\rbrace  \in \mathcal{F}$$
> を満たすものを **（ $S$値の）確率変数** と言う。

#### 集合の記法 $\lbrace ... : ...\rbrace$

$\lbrace x \in S : P(x)\rbrace$ は **「 $S$ の元 $x$ のうち、条件 $P(x)$ を満たすもの」の集合**（内包的記法）。コロン `:` は "such that"（「〜を満たすような」）。
よって $\lbrace \omega \in \Omega : X(\omega) \in G\rbrace$ は「 $X(\omega)$ が $G$ に入る $\omega$ をすべて集めた集合」。慣用的には $\lbrace X \in G\rbrace$ と略す。

#### 条件の意味

「 $X$ の値がある集合 $G$ に入る」という $\Omega$ 側の事象 $\lbrace \omega : X(\omega) \in G\rbrace$ が、ちゃんと $\mathcal{F}$ で測れる必要がある、ということ。

- $\mathcal{F}$ で測れない = 確率が定義されない = $X$ が確率的なものとして使えない

つまり「**$X$ がどんな値の集合に入る確率も、ちゃんと測れる**」という保証が確率変数の条件。

#### 逆像の記法

$X^{-1}(G) := \lbrace \omega \in \Omega : X(\omega) \in G\rbrace$ と書くと、条件は

$$
G \in \mathcal{G} \ \Rightarrow\  X^{-1}(G) \in \mathcal{F}
$$

「**$X$ で $G$ を $\Omega$ に引き戻したら $\mathcal{F}$ の元になる**」という気持ち。

## 実数値の場合は条件が簡単

> **定義 1.7**（実数値確率変数）確率空間 $(\Omega, \mathcal{F}, P)$ に対し、写像 $X : \Omega \to \mathbb{R}$ であって、任意の $a \in \mathbb{R}$ について
> $$\lbrace \omega \in \Omega : X(\omega) \le a\rbrace  \in \mathcal{F}$$
> を満たすものを **（実数値の）確率変数** と言う。

#### 行き先の可測空間は何？

実数値の場合、行き先の可測空間は **暗黙のうちに $(\mathbb{R}, \mathcal{B}(\mathbb{R}))$** が採用されている:

- $\mathbb{R}$: 実数全体
- $\mathcal{B}(\mathbb{R})$: $\mathbb{R}$ 上のボレル集合族（すべての開区間を含む最小のσ加法族）

本文でも「通常、すべての開区間を含む最小の σ加法族、いわゆるボレル集合族 $\mathcal{B}(\mathbb{R})$ をとる」と書かれている。

#### なぜ条件が「 $\lbrace X \le a\rbrace  \in \mathcal{F}$」だけで済むのか

定義1.6（一般）では「任意の $G \in \mathcal{G}$ について $X^{-1}(G) \in \mathcal{F}$」だった。実数値で素直に当てはめると「任意の $B \in \mathcal{B}(\mathbb{R})$ について $X^{-1}(B) \in \mathcal{F}$」となるはず。
ところが定義1.7では **半無限区間 $(-\infty, a]$ について** だけでよい、となっている。理由:

- ボレル集合族 $\mathcal{B}(\mathbb{R})$ は **半無限閉区間 $(-\infty, a]$ から生成される**（開区間生成と同値）
- 逆像 $X^{-1}$ は集合演算と交換する（ $X^{-1}(A^c) = X^{-1}(A)^c$, $X^{-1}(\bigcup A_i) = \bigcup X^{-1}(A_i)$）
- なので生成元 $(-\infty, a]$ で $X^{-1}((-\infty, a]) = \lbrace X \le a\rbrace  \in \mathcal{F}$ を確認すれば、ボレル集合族のすべての元について自動的に $X^{-1}(B) \in \mathcal{F}$ が成り立つ

#### $\le a$ じゃなく $\ge a$ ではダメ？

理論的には **どちらでも同じ**。 $\le a$ なのは数学的必然ではなく慣習。

ボレル集合族は、以下のどれを生成元に選んでも作れる:

| 生成元 | 結果 |
|---|---|
| $(-\infty, a]$ for all $a$ | ✓ ボレル集合族 |
| $[a, \infty)$ for all $a$ | ✓ ボレル集合族 |
| $(-\infty, a)$ / $(a, \infty)$ | ✓ ボレル集合族 |
| 開区間 $(a, b)$ | ✓ ボレル集合族 |
| 閉区間 $[a, b]$ | ✓ ボレル集合族 |

例えば $\lbrace X \le a\rbrace  \in \mathcal{F}\ \forall a$ から $\lbrace X \ge a\rbrace  \in \mathcal{F}\ \forall a$ は次のように導ける:

$$
\lbrace X < a\rbrace  = \bigcup_{n=1}^{\infty} \lbrace X \le a - \tfrac{1}{n}\rbrace  \in \mathcal{F} \quad\text{(可算和)}
$$
$$
\lbrace X \ge a\rbrace  = \lbrace X < a\rbrace ^c \in \mathcal{F} \quad\text{(補集合)}
$$

逆方向も同様。なので

$$
\lbrace X \le a\rbrace  \in \mathcal{F}\ \forall a \ \ \Longleftrightarrow\ \  \lbrace X \ge a\rbrace  \in \mathcal{F}\ \forall a
$$

$\le a$ を選ぶのは慣習的な理由（累積分布関数 $F(a) = P(X \le a)$ との整合性、「 $a$ 以下」の日常的書きやすさなど）。

#### まとめ

| 定義 | $X$ の行き先 | 行き先の可測空間 | 確認すべき条件 |
|---|---|---|---|
| 定義1.6（一般） | 任意の集合 $S$ | $(S, \mathcal{G})$ | 任意の $G \in \mathcal{G}$ について $X^{-1}(G) \in \mathcal{F}$ |
| 定義1.7（実数値） | $\mathbb{R}$ | $(\mathbb{R}, \mathcal{B}(\mathbb{R}))$ | 任意の $a \in \mathbb{R}$ について $\lbrace X \le a\rbrace  \in \mathcal{F}$ |

実数値の場合、行き先の可測空間が $(\mathbb{R}, \mathcal{B}(\mathbb{R}))$ で固定され、確認も「半無限区間だけ」でよい、というのが定義1.7のお得な点。

## 記号の慣習

「 $X$ の値が集合 $A$ に入る確率」を

$$
P(X \in A) := P(\lbrace \omega \in \Omega : X(\omega) \in A\rbrace )
$$

と略記する。

→ 入門書で「 $P(X \in A)$」「 $P(X \le a)$」「 $P(X = c)$」と書かれているとき、その背後にこの仕掛けが効いている。

## 1.2.5 のまとめ

- **確率変数** = 確率空間 $(\Omega, \mathcal{F}, P)$ から値の空間への写像で、 $\lbrace X \in G\rbrace$ という事象が常に $\mathcal{F}$ で測れるもの
- **実数値の場合** は $\lbrace X \le a\rbrace$ がすべての $a$ について測れればOK
- 記法 $P(X \in A)$ の背後には、逆像 $X^{-1}(A)$ が $\mathcal{F}$ の元であることが暗黙に効いている
