# 1.2.2 確率の抽象的定義（一般の場合）

> 該当ページ: 13–16（[book/0023.jpg](../book/0023.jpg)–[book/0026.jpg](../book/0026.jpg)）

## 3行サマリ

1. $[0,1]$ の素朴な確率定義は破綻する → 確率を測れる集合を $2^\Omega$ より狭く絞る
2. **σ加法族** $\mathcal{F}$ = 補集合と可算和で閉じた集合族（確率を測れる集合のリスト）
3. **確率測度** $P$ は $\mathcal{F}$ 上で $P(\Omega)=1$ + 可算加法性を満たす写像

**この章で最大の山場**。連続値（ $\Omega = [0,1]$ など）に対応するため、確率を定義する集合の範囲を制限する仕組みを導入する。

## なぜ抽象化が必要なのか

1.2.1 の定義を $\Omega = [0,1]$ にそのまま持ち込もうとすると **破綻する**。

#### 破綻の例

- 一点 $\lbrace a\rbrace$ の確率を $0$ とすると、和の法則で全事象の確率も $0$ になる
- $\lbrace a\rbrace$ の確率を $0$ より大きくとると、点が無限個あるので合計が無限大に発散
- 有理数集合 $S$ と無理数集合 $T$ で $S \sqcup T = [0,1]$ だが、両者に「区間の長さ」のような自然な確率を与える方法がない

→ **すべての部分集合に確率を割り振ろうとするのが間違い**。確率を定義できる集合の範囲を最初から制限する。これが測度論の発想。

---

## ステップ1: σ加法族 — 「確率を測れる集合の族」

確率を定義する集合の範囲を、 $2^\Omega$ より狭い「都合の良い集合族」 $\mathcal{F}$ にする。この $\mathcal{F}$ が満たすべき条件:

> **定義 1.2** $\Omega$ の部分集合の族 $\mathcal{F}$ が **σ加法族** とは:
> 1. $\emptyset \in \mathcal{F}$
> 2. $A \in \mathcal{F}$ なら $A^c \in \mathcal{F}$（補集合で閉じる）
> 3. $A_1, A_2, \ldots \in \mathcal{F}$ なら $A_1 \cup A_2 \cup \cdots \in \mathcal{F}$（**可算個** の和集合で閉じる）

ペア $(\Omega, \mathcal{F})$ を **可測空間** と呼ぶ。

#### 「可測」って何が「測れる」のか

「可測 (measurable)」 = **測度（長さ・確率など）を入れることができる**、という形容。

可測空間 $(\Omega, \mathcal{F})$ で「測れる」のは **$\mathcal{F}$ の元（可測集合）**。「ここに測度を入れる準備ができている」という枠組みを表す。

ただし、可測空間の段階では **まだ測度 $P$ は入っていない**。入れると一段上の構造になる:

| 構造 | 名前 |
|---|---|
| $(\Omega, \mathcal{F})$ | **可測空間** — 測れる準備ができた状態 |
| $(\Omega, \mathcal{F}, P)$ で $P$ が確率測度 | **確率空間** — 確率が入った |
| $(\Omega, \mathcal{F}, \mu)$ で $\mu$ が一般の測度 | **測度空間** — 測度が入った |

例えるなら:
- 可測空間 = 区画整理が終わって「ここに建物を建ててよい」と決まった土地
- 確率空間 = そこに実際に建物（確率測度 $P$）を建てた状態

### この定義は $2^\Omega$ の "代替物" になる

本書はこのあと「σ加法族は有限の場合の $2^\Omega$ の代替物になりそう」と書く。これは **「σ加法族 ≒ $2^\Omega$」と言いたいのではなく、「 $2^\Omega$ が有限の場合に果たしていた役割を、σ加法族が果たせる」** という意味。

#### $2^\Omega$ が有限の場合に嬉しかった理由

定義 1.1 で $P : 2^\Omega \to [0,1]$ と置いたのはなぜか。

- $A, B \in 2^\Omega$ なら $A \cup B, A \cap B, A^c$ もすべて $2^\Omega$ の元
- つまり **集合演算で閉じている**
- これが効くのは、 $A, B$ が定義域に入っていれば、 $A \cap B$ や $A \cup B$ といった派生事象も自動的に定義域に入り、 $P(A \cap B)$ や $P(A \cup B)$ という記号が **意味を持つ（値が確定する）** という保証になること
- 注意: 「派生事象の確率値が $P(A), P(B)$ から計算できる」という意味ではなく、「派生事象の確率が **定義されている (well-defined)**」という、もっと手前の話。値そのものは集合の中身に依存

→ $2^\Omega$ の役割は **「事象をどう組み合わせても、その派生事象に対しても確率が定義されている」を保証すること**。

#### σ加法族はこの役割を引き継ぐ

σ加法族の3条件 + ド・モルガンの法則 $A \cap B = (A^c \cup B^c)^c$ から、**集合演算（∩, ∪, $^c$）が自由にできる**。これは $2^\Omega$ がやっていた仕事そのもの。

#### 有限の場合は $2^\Omega$ 自身がσ加法族

$\Omega$ が有限なら $2^\Omega$ は3条件をすべて満たすので、 $2^\Omega$ もσ加法族の一例。
つまり **σ加法族は $2^\Omega$ の一般化**:

- 有限の場合: $\mathcal{F} = 2^\Omega$ で従来通り
- 無限・連続の場合: $2^\Omega$ 全体は使えないが、より狭いσ加法族 $\mathcal{F}$ を取れば、同じ「集合演算で閉じている」性質を保ったまま確率が定義できる

これが「代替物」の意味。

### σ加法族の具体例（ $\Omega = \lbrace 1, 2, 3, 4, 5, 6\rbrace$ サイコロ）

**例A: 自明なσ加法族** $\quad \mathcal{F}_0 = \lbrace \emptyset, \Omega\rbrace$
- 「何かが起こる」「何も起こらない」しか区別できない、最も粗いリスト

**例B: 丁か半か（偶数か奇数か）だけ気にする** $\quad \mathcal{F} = \lbrace \emptyset, E, E^c, \Omega\rbrace$（ $E = \lbrace 2,4,6\rbrace$）
- 4つの集合で集合演算が閉じている
- 細かい目は気にせず、丁半だけ知りたい問題ならこれで十分（本書の例1.4）

**例C: 最大のσ加法族（冪集合）** $\quad \mathcal{F} = 2^\Omega$
- $2^6 = 64$ 個の部分集合すべて
- 有限の場合の「フル装備」

→ σ加法族は **「自分が気にする事象たちが、集合演算で閉じるようにまとめたリスト」**。

## ステップ2: 確率（測度）の定義

> **定義 1.3** 可測空間 $(\Omega, \mathcal{F})$ 上の確率 $P : \mathcal{F} \to [0, 1]$ とは:
> 1. $P(\Omega) = 1$
> 2. **可算加法性**: 互いに素な $A_1, A_2, \ldots \in \mathcal{F}$ について
>    $$P(A_1 \sqcup A_2 \sqcup \cdots) = \sum_{n=1}^{\infty} P(A_n)$$

3つ組 $(\Omega, \mathcal{F}, P)$ を **確率空間** と呼ぶ。本書および現代確率論の最も基本的な舞台。

### 確率空間の3点セット — 何が入っているか

| 部品 | 何者 | 役割 |
|---|---|---|
| $\Omega$ | 全体集合 | ありうる結果すべての集合 |
| $\mathcal{F}$ | σ加法族 | 確率を測れる事象（ $\Omega$ の部分集合）のリスト |
| $P$ | 確率測度 | 各事象 $A \in \mathcal{F}$ に $[0,1]$ の値を割り当てる写像 |

3つそろって初めて「確率を計算できる舞台」になる。

### 具体例

#### 例1: コイン投げ
- $\Omega = \lbrace H, T\rbrace$
- $\mathcal{F} = 2^\Omega = \lbrace \emptyset, \lbrace H\rbrace , \lbrace T\rbrace , \lbrace H, T\rbrace \rbrace$
- $P(\lbrace H\rbrace ) = P(\lbrace T\rbrace ) = 1/2$

#### 例2: サイコロ
- $\Omega = \lbrace 1, 2, 3, 4, 5, 6\rbrace$
- $\mathcal{F} = 2^\Omega$（64個の部分集合）
- $P(\lbrace i\rbrace ) = 1/6$

#### 例3: $[0, 1]$ 上にランダムに一点を選ぶ
- $\Omega = [0, 1]$
- $\mathcal{F} = $ ボレル集合族（閉区間を含む最小のσ加法族）
- $P([a, b]) = b - a$（区間の長さ）

### 確率空間の使い方（サイコロ）

「奇数が出る」事象 $A = \lbrace 1, 3, 5\rbrace  \in \mathcal{F}$ について:

$$
P(A) = P(\lbrace 1\rbrace ) + P(\lbrace 3\rbrace ) + P(\lbrace 5\rbrace ) = \tfrac{1}{6} + \tfrac{1}{6} + \tfrac{1}{6} = \tfrac{1}{2}
$$

$\Omega$ から事象 $A \in \mathcal{F}$ を取り出して、 $P$ で値を計算する。これが確率空間の "使い方"。

### ポイント

- 1.2.1 の有限和の法則が、**可算無限** へ拡張されている
- これが連続値の確率を扱える数学的鍵

> **重要な一言**: この公理さえ満たせば数学的には確率。
> **「不確実な出来事に対する主観的な見積もり」も、これらの条件を満たせば確率である** — これがベイズ確率を許す数学的な根拠。

## 補足: 「有限個」「可算個」「非可算個」の違い

σ加法族の "σ" は条件3の **可算個** に由来する。

| 用語 | 意味 | 例 |
|---|---|---|
| **有限個** | 個数が具体的な自然数 $n$ で抑えられる | $\lbrace 1, 2, 3\rbrace$、サイコロの目 |
| **可算個** (countable) | **自然数 $\mathbb{N} = \lbrace 1, 2, 3, \ldots\rbrace$ と一対一対応がつく** | $\mathbb{N}$、整数 $\mathbb{Z}$、有理数 $\mathbb{Q}$ |
| **非可算個** (uncountable) | 可算個ではない | 実数 $\mathbb{R}$、区間 $[0, 1]$ |

ポイント:
- 「可算個」は「有限個」と「可算無限個」の両方を含む。 $A_1, A_2, \ldots$ と番号付けて列挙できる、というのが本質
- 有理数 $\mathbb{Q}$ は無限集合だが **可算**
- 実数 $\mathbb{R}$ は **非可算**（カントールの対角線論法）

#### なぜ "可算個" がちょうどいいのか

**有限個では弱すぎる**:
- 連続値の確率では極限操作が要る。例: $[0, 1) = \bigcup_{n=1}^{\infty} [0, 1 - \tfrac{1}{n}]$ は **可算無限** の和集合
- 有限個しか許さないと、こうした極限操作で集合族からはみ出してしまう

**非可算個まで許すと壊れる**:
- $[0,1]$ の任意の部分集合は各点 $\lbrace a\rbrace$ の和として書ける
- 非可算和まで許すと、 $2^{[0,1]}$ と同じになり、確率を定義できない部分集合まで含んでしまう

**可算個がちょうどいい**:
- 確率論で必要な極限操作はすべて可算回で済む
- 病的な部分集合は除外できる

→ "σ" は **「ちょうど可算個まで許す」絶妙なバランス**。
